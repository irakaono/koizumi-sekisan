## 2. Folder Explorer（最優先実装）

### UI構成
```
┌─────────────────┬─────────────────────────────────┐
│  Folder Tree    │  Item List                      │
│  （左ペイン）    │  （右ペイン）                    │
│                 │                                 │
│ 📁 外構  163行  │  外構 › 舗装                    │

│   📁 サイン基礎 │  1工区 アスファルト ¥231,099    │
│     📁 サイン2  │  路床整正           ¥200,232    │
│     📁 サイン11 │  ...                            │

└─────────────────┴─────────────────────────────────┘
```

### 左ペイン（Folder Tree）仕様

- クリックで選択・展開/折りたたみ
- ドラッグ&ドロップで順序変更（→ 自動でRevision生成）
- 右クリックメニュー：
  - フォルダ名変更
  - フォルダ移動
  - フォルダ削除
  - ルールとして保存

### 右ペイン（Item List）仕様

- 選択フォルダ内の明細を一覧表示
- カラム：明細名 / 数量 / 単位 / 設計金額 / 実行予算単価 / 備考
- 行クリックで編集可能
- 右クリックで：フォルダ移動 / ルールとして保存

### 警告表示

| 状態 | 表示 |
|------|------|
| フォルダ分断 | 🔴 フォルダ名が赤字、⚡マーク |
| フォルダ重複 | 🔴 ⚠マーク付き |
| 金額異常 | 🟡 行に⚠マーク |
| 空フォルダ | 🔵 フォルダ名がグレー |

## 4. フォルダツリー描画仕様

### 3階層の表示ルール

```
F1（親フォルダ）    → 青色  #58a6ff  太字
  F2（子フォルダ） → 緑色  #7ee787
    F3（明細名）   → オレンジ #f0883e  小フォント（先頭3件のみ）
              ↑ 「…他N件」で省略
```

### 分断検出ロジック
```javascript
// F1レベルの分断検出
let cur = null, seen = new Set(), broken = new Set();
rows.forEach(r => {
  if (r.f1 !== cur) {
    if (seen.has(r.f1)) broken.add(r.f1);  // 再出現 = 分断
    seen.add(r.f1);
    cur = r.f1;
  }
});
```

## 5. リアルタイム更新

Rule変更 → Simulator即時反映の実装方針：

```javascript
// Rule変更のたびに呼ぶ
function onRuleChanged() {
  const processed = applyRules(allRows, currentRulePackage);
  updateFolderExplorer(processed);
  updateSimulatorPanel(processed);
  runValidator(processed);
}
```
